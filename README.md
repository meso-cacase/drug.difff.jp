くすりのリスト - 医薬品の簡易検索サイト
======================

日本の医療用医薬品およびOTC医薬品を簡単に検索することができるサイトです。  
販売名や一般名をインクリメンタル検索して高速に表示します。

ページをHTML形式でダウンロードしてそのファイルをブラウザで開いても、  
本家サイトと同様に検索することができます。

本ソフトウェアはオープンソースです。どなたでも無償で自由に利用できます。  
**くすりのリスト** 稼働中： https://drug.difff.jp/
